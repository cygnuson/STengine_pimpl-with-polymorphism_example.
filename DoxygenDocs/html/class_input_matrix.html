Detailed Description

The input matrix` sole job is to record user input from the keyboard, the mouse, and the joystick.

The InputMatrix is designed to be called LAST after all other input has been checked for. This makes sure that all event are handled. If you (the dev) want the non-input events to be ignored, use IgnoreBadInput(true);
